# Sketch 81.1 (135647)

> This Homebrew Tap provides a Cask for Sketch 81.1

## Requirements

### Tools

- [Homebrew](https://brew.sh)

## Installation

### 1. Setup the Tap

```shell
brew tap pascaliske/sketch-81
```

### 2. Install Sketch

```shell
brew install pascaliske/sketch-81/sketch-81
```
